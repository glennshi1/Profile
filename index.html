Ideal Job

Software Engineer in Test at Electronic Arts

Job Description

This position, Software Engineer in Test, often shortened to SET, is sort of like a software developer who develops software for testing. They build tools, frameworks and automated tests which can be used to run tests rapidly and repeatedly. Due to the nature of the SET role, in recent years they have largely replaced the traditional manual testing role (such as QA), although there is always some manual testing needed for problems that a human can catch more easily than a test script. This position is particularly appealing to me since I would like to have a career somewhere in game development and because it is in the Melbourne city centre.

Project Idea

Overview

My project idea is to develop a game using Unity. The project will be a platform type game (or platformer) and keeping into consideration that a complete game could take an extraordinary amount of time and effort which would be unfeasible, the project will aim to produce a demo version. In addition, the experience using a game engine like Unity and creating a game with it could also help me develop some of the skills and knowledge required to make me qualified for my ideal position as a Software Engineer in Test.

Motivation

This project will realistically aim to be interesting and entertaining for the user, whomever it may be. That being said, a possible future revision could see it entering the gaming market, one of the fastest growing sectors of present. The global gaming market was valued at USD 162.32 billion in 2020 and is expected to reach a value of USD 295.63 billion by 2026. While there is a massive market for gaming, it is also highly competitive and dominated by big names and companies.

Description

The project will be a platform action, side scroller game. The overall setting and theme of the game and its elements will be based on a story; however, the story will have little importance in the actual gameplay which will be the primary focus. As this is my first time creating such a project, there will almost certainly be some alterations and changes to some of the described features and components at a future date.

The main objective for the player will be to progress, moving across from left to right, where there will be an end goal. The player will have to move and jump across platforms and figure out their way past obstacles and enemies. The player will have movement such as left, right, and jump. Movement will come from arrow key inputs from the user. The player will also have an attack function to use against enemies which will likely come from the spacebar key.

There will be sound effects in the game, based on certain actions, events and interactions with the environment. The game will if possible, also include an original soundtrack or OST, although since I have no experience creating music this will most likely require the skillset of another person. The game will likewise have objects, sprites and backgrounds which will preferably be original. Once again as creating art and drawing whether on computer or hand drawn, is not one of my talents, this will most likely require the skills of another person. Having original art and music will be crucial in making the game as unique as possible, considering that there are an endless number of games already published.

The game will be made for PC, although there could be a possible mobile version made later. Eventually the game will be released through Steam, a video game digital distribution service where it can reach a wide audience and be distributed globally.

Tools and Technologies

In order to develop the project, the essential hardware required is a PC and the relevant external components, such as a monitor, mouse, keyboard, etc. Otherwise, a decent laptop or something similar that can run the required software can be used. Required software include Unity, or possibly another game engine such as Unreal, a digital audio workstation such as Studio One, and a digital art software like Adobe Photoshop. Access to the internet and being able to communicate with other people working on the project is also essential.

Skills Required

Project management would be important to lead the work of the team, to keep us on track and to meet our goals within the time limits. Experience creating a game with a game engine and having the essentials of programming will be an important skill for this project. Knowing these will make the majority of developing the game much easier. The project also requires the knowledge and skills of creating art using a digital art software. In addition, having the ability to use a digital audio workstation (DAW) and being able to create audio and soundtracks will be a necessary requirement for the project. Acquiring the skills and skillsets required should be reasonably possible.

Outcome

The project’s goal to provide entertainment and enjoyment for the user. Although not outside the realms of possibility, selling and making profit is not a core aim. If the user enjoys playing the game that this project produces, then it can be considered successful. This project’s main impact will be on the team behind it and the developers, who will expectantly have furthered their understanding of game development, programming, using various software and IT.
